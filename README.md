# Kintone PC/Mobile Switcher

kintoneのPC版とモバイル版を切り替えるChrome拡張機能

## 機能

kintoneのページを開いた状態で拡張機能アイコンをクリックすると、対応するPC版/モバイル版のページに切り替えられる

## インストール

[Chrome Web Store](https://chromewebstore.google.com/detail/kintone-pcmobile-switcher/pngkjmlmldlofcoeaafjidonobelclal)
